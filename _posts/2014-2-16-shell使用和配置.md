reset ( a shell cmd )
    因为某些原因导致终端的输出完全乱掉的时候，连命令提示符都是乱码，输入reset然后回车。当然，此时输入的reset也无法正常回显，但它确实正常工作。

蓝色注释看不清

文件扩展名与打开方式
    `alias -s txt=vi`

    `% a\_filename.txt\` # 仅仅输入一个\*.txt文件名,就会自动用vi打开

    还可以`alias -s pdf=evince`...
