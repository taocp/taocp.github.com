Linux和Linux共享文件?

网上多为Linux和windows共享的介绍. 主机和客户机都是Linux少见.

以ubuntu 12.04作客户机 和 vbox 4.2.22为例.

  1. 在vbox ---> settings ---> shared folders中添加待共享的路径,为该路径指定一个名字(Folder Name)

  2. 启动客户机,在主机的vbox菜单栏选择 Devices ---> Install Guest Additions(Host+D). 安装该程序,不同发行版可能有不同方法.

  3. 在客户机中打开终端, 输入sudo mount -t vboxsf -o uid=1000,gid=1000 _folder_name_ _your_mount_point_

    uid=1000,gid=1000 : 1000指当前用户的id, 可以输入id命令查看,将1000替换为自己的id.

    _folder_name_ : 第一步中为共享路径指定的名字.

    _your_mount_point_ : 挂载点.

  4. 举例:

    • 共享主机上/home/happy目录,指定名字为ohappy;
    • 安装Guest Additions
    • 计划将主机的/home/happy挂载到客户机中当前目录下的hostd目录上
    • 在客户机中输入sudo mount -t vboxsf -o uid=1000,gid=1000 ohappy hostd
